# Changelog

## [0.1.0] - 2026-07-11

### Added
- Initial release
- Multi-source boundary query: OSM, Baidu Maps, Amap
- Coordinate transformation: BD-09 MC / BD-09 / GCJ-02 / WGS84
- CLI tool: `geoboundary "place name"`
- Python API: `get_boundary()`, `batch_query()`
- Export formats: GeoJSON, Shapefile, CSV/WKT
- Pre-built dataset: 1013 Chinese National/Provincial Scenic Areas (in GitHub Releases)
